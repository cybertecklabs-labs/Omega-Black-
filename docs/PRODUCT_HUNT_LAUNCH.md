**Q: How is this different from PentestGPT?**

**Q: Is this safe to run?**
A: Yes. The AEVC (Autonomous Exploit Validation Cycle) is engineered to perform *safe* interactions. It verifies vulnerabilities without destructive payloads (e.g., verifying SQLi logic without dropping tables).

**Q: Can I run this offline?**
A: 100%. That is the core of our "Sovereign" promise. You can perform air-gapped deployments using local LLMs (Llama 3 / Mistral) via Ollama.
